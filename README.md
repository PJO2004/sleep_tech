# TeamName: 나비잠
## Environment
- .github: Github action을 동작하기 위한 폴더입니다.
- backend: 시스템 구성파일을 관리합니다.
- frontend : templates나 static과 같은 웹페이지를 구성합니다.
- Dockerfile: Docker 플랫폼에서 실행될 이미지를 정의합니다.
- requirements: 라이브러리 정보를 가지고 있습니다.
